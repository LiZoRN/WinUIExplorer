由于facebook官方是用mac系统，提供的example无法在windows下运行，固进行移植，在windows下编译和运行本工程需要如下操作：
请安装好npm,nodejs,react-native基本配置

1.在工程目录下npm install

2.node node_modules/react-native/packager/packager.js 

3.在开启安卓虚拟机或者，react-native run-android

4.打包 http://facebook.github.io/react-native/docs/signed-apk-android.html#content
